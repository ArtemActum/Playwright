## Prerequisites
- Node.js (v16 or higher)
- npm (comes with Node.js)
- Git

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/ArtemActum/Playwright.git
   cd Playwright
2. Clone the repository:
   
   ```bash
   npm install
3. To run all Playwright tests:

   ```bash
   npx playwright test
